## New <<- Operator

```
a<-1
b<-2
doit<-function(x)
{
    a<-3
    print(sprintf("a:%d x:%d",a,x))
}
doit(b)
a
```
Which yields:

```
> doit(b)
[1] "a:3 x:2"
> a
[1] 1
```

Let's try this with the <<- operator...
```
a<-1
b<-2
doit<-function(x)
{
    a<<-3
    print(sprintf("a:%d x:%d",a,x))
}
doit(b)
a
```

This yields:

```
> doit(b)
[1] "a:3 x:2"
> a
[1] 3
```

Note the <<- changed the value of a in the calling environment, not just in the environment local to the function.
